# COMO EXECUTAR O PROJETO

## Instalação e downloads necessários:
1. JDK 8 ou superior
2. Maven 3.6.3
3. Eclipse IDE
4. ChromeDriver.exe

## Passo a passo para a execução:
1. Realizar o clone do projeto na IDE
2. Entrar na Pasta "src/test/java"
3. Acessar o Package "utils"
4. Na classe "TestRunner.java", clicar com o botão direito e execultar os testes em: "Run As >> Junit Test"

## Relatório da execução dos testes:
1. Acessar a Pasta "target"
2. Acessar a Pasta "report-html"
3. Abrir o arquivo "index.html"
